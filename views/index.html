Timestamp Microservice

How It Works

Pass a string to the URL. If it contains either a Unix timestamp or a natural language date, the service will return a JSON object containing both the timestamp and natural language form of the given date. If the string is not a valid date, the service will return the same object with null values.

Examples

  • 1504656000
  • September 6, 2017
  • 06 Sep 2017
  • 09 06 17
  • not a date value
